# headway-core

Core library for [Headway](../README.md): GTFS parsing, validation, CRUD,
and writing. No CLI concerns, no `process::exit`, no `println!`.

This crate is consumed by the [`headway` CLI binary](../cli) and is not
intended for standalone use. See the [top-level README](../README.md) for
installation, usage, and project overview.
